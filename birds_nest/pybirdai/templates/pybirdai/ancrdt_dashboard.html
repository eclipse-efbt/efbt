About ANCRDT Transformation

The ANCRDT (Anacredit) transformation process converts regulatory data models into executable transformations for ANACREDIT reporting. This four-step process fetches CSV data from the ECB website, imports ANCRDT data, creates join metadata, and generates executable Python code for data transformations.
