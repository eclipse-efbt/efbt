CSV File Format Requirements:

  • The file must be in CSV format with UTF-8 encoding
  • Column headers must follow the pattern:
    • SOURCE_VARIABLE_1, SOURCE_VARIABLE_2, ... for source variables
    • TARGET_VARIABLE_1, TARGET_VARIABLE_2, ... for target variables
    • SOURCE_MEMBER_1, SOURCE_MEMBER_2, ... for source members
    • TARGET_MEMBER_1, TARGET_MEMBER_2, ... for target members
    • MEMBER_HIERARCHY, VALID_FROM, VALID_TO (optional)
  • Each row represents one complete mapping rule
  • All variable and member IDs must exist in the database
  • Dates should be in YYYY-MM-DD format
